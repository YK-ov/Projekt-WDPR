Algorym 8 - sortowanie bąbelkowe (Bubble Sort). Chociaż nie jest to najbardziej wydajny algorytm,
można go zrównoleglić, dzieląc iteracje na różne wątki. Niemniej jednak, jego złożoność
czasowa sprawia, że równoleglenie jest bardziej użyteczne w małych zbiorach
danych. Całkowita złożoność czasowa tego algorytmu wynosi od O(n) do O(n^2).
